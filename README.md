# IPM — Information Process Management

A connected BIM information-management prototype linking owner outcomes, use cases, processes, information requirements, roles, tools, validation and FM handover.

## Product routes

- `/` — interactive information graph and method
- `/knowledge/` — eight-part knowledge series
- `/standards/` — Swiss standards implementation workbench
- `/tables/` — structured source tables

## Swiss standards layer

The workbench maps ISO information management, Swiss LOIN guidance, BIM-Abwicklungsmodell, UCM, CRB Basis-AIR, BIM2FM, eBKP–IFC, IFC, IDS and bSDD to project applicability, responsibility, evidence and readiness.

It contains reference metadata and implementation guidance only. Licensed standards and protected datasets are not reproduced.

See [`docs/SWISS_STANDARDS_LAYER.md`](docs/SWISS_STANDARDS_LAYER.md) for the product model and delivery roadmap.
